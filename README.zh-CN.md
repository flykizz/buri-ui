## 特性

* 60+ 高质量组件
* 90% 单元测试覆盖率
* 完善的中英文文档和示例
* 支持按需引入
* 支持主题定制
* 支持国际化
* 支持 TS
* 支持 SSR

## 安装

```bash
# 通过 npm 安装
npm i buri -S

# 通过 yarn 安装
yarn add buri
```

## 快速上手

```js
import Vue from 'vue';
import { Button } from 'buri';
import 'buri/lib/index.css';

Vue.use(Button);
```

buri 也支持按需引入、CDN 引入等方式，详细说明见 [快速上手](https://flykizz.github.io/buri#/zh-CN/quickstart).
